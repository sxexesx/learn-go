# Какие основные фичи и особенности появились в версиях golang

## 1.22

1. Исправление поведения переменных цикла for. В Go 1.22 каждая итерация создает новые переменные, что решает классическую проблему с goroutines

2. Range over integers
```golang
for i := range 10 {
    fmt.Println(10 - i)
}
```

3. Новый пакет `math/rand/v2` - новые методы и функции:
- глобальный генератор теперь инициализируется случайным значением
- интерфейс упрощен до одного метода
```
randomUint := rand.UintN(100)  
```
- более быстрые алгоритмы для криптографической надежности ChaCha8, PCG

4. Улучшенная HTTP маршрутизация в net/http.ServeMux (см. пример)

5. В пакете `slices` добавилась функция `concat`
```golang
s1 := []int{1,2}
s2 := []int{3,4}

result := slices.Concat(s1, s2)
```

6. Новый пакет go/version

7. GC оптимизация
Runtime теперь хранит GC метаданные ближе к объектам в heap, улучшая производительность на 1-3% и снижая пасять на ~1% 

8. Profile-guided Optimization (PGO)
Компилятор теперь лучше использует PGO для девиртуализации вызовов методов интерфейсов, обеспечивая прирост производительности на 2-14%.

9.  Улучшения Vet инструмента

10. Полная переработка execution tracer с поддержкой stream-обработки, корреляцией с системным временем и полными длительностями системных вызовов
